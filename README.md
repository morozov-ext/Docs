# Сервер **weaselcloud-26445**

## Описание
Сервер предоставляет веб-окружение на основе **LEMP-стека** и набор базовых сервисов.  
Обслуживает HTTP-запросы через **nginx**, исполняет PHP-скрипты через **php-fpm**, использует **MariaDB** в качестве реляционной СУБД.

В каталоге `/var/www/testapp` размещены файлы приветственной страницы:
- `index.html`
- два PNG-изображения

## Назначение
- Тестовый стенд («песочница») для кандидатов, знакомящихся с веб-окружением.
- Отображение приветственной страницы при обращении к серверу.

## Логические связи
- **nginx** принимает HTTP-запросы, отдаёт статические файлы и проксирует динамические в **php-fpm**.
- **php-fpm** при выполнении PHP-кода взаимодействует с:
  - **MariaDB** — `127.0.0.1:3306`
  - **Memcached** — `127.0.0.1:11211`
- **Postfix** обслуживает локальные системные уведомления.
- **Fail2Ban** анализирует логи и блокирует подозрительную активность (особенно — попытки bruteforce через SSH).
